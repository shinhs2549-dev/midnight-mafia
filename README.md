# 🌙 미드나잇 마피아 (수학여행용 스마트폰 마피아 게임)

선생님(사회자)이 진행을 컨트롤하고, 학생들은 각자 폰으로 접속해서 하는 실시간 마피아 게임입니다.
밤에는 **살아있는 전원의 폰이 켜져서** 다 같이 익명 채팅을 하기 때문에, 폰만 봐서는 누가 마피아인지 알 수 없습니다.

## 게임 흐름 한눈에
1. 사회자가 폰/노트북에서 접속 → **사회자로 방 만들기** → 방 코드(4자리) 생성
2. 학생들은 같은 주소 접속 → 이름 + 방 코드 입력 → 입장
3. 사회자가 인원 보고 직업 구성 맞추고(자동 추천 버튼 있음) → **게임 시작**
4. 각자 폰에 역할(마피아/경찰/의사/시민) 표시
5. **밤** → 전원 폰 켜짐. 마을 익명채팅 + 지목. 마피아는 마피아 채팅 추가. (타이머)
6. 사회자가 **밤 종료** → 누가 죽었는지 발표 + 마을 여론(참고용) 공개
7. **낮** → 다 같이 입으로 토론 → 사회자가 처형 대상 선택
8. 마피아 전멸(시민 승) 또는 마피아 수 ≥ 시민 수(마피아 승)까지 반복

## 핵심 규칙(우리가 정한 대로 구현됨)
- **연막**: 밤에 마피아·경찰·의사뿐 아니라 **시민도 모두** 지목 화면 + 채팅을 봅니다.
- **마을 채팅**: 살아있는 전원이 **익명**으로 대화(매 밤 임시번호 재배정). 마피아도 시민인 척 섞입니다.
- **마피아 채팅**: 마피아끼리만 보이는 별도 채팅.
- **마을 여론 투표**: 시민들의 지목만 집계해 낮에 "참고용"으로 공개. 마피아의 지목은 살해 투표라 여론 집계엔 들어가지 않습니다.
- **토론 마중물**: 매 밤 무작위로 2~3명을 "오늘의 화제"로 띄워서 시민 밤채팅이 붕 뜨지 않게 합니다.
- **인원 유동 관리**: 시작 전 실시간 접속자 수 + 직업 +/- 조정 + 합 검증. 게임 중에는 플레이어를 눌러 *사망 처리* 또는 *완전 퇴장* 가능. 접속 끊김은 회색 점으로 표시되고, 같은 이름으로 재접속하면 역할이 복구됩니다.
- **밤 길이 타이머**: 기본 60초(20~300초 조절). 전원 동시에 시작/종료.

> 27명 기준 자동 추천 구성: **마피아 6 · 경찰 2 · 의사 1 · 시민 18**
> (인원이 줄면 자동 버튼이 비율에 맞게 다시 추천해 줍니다.)

## 실행 방법

### A. 가장 쉬운 배포 — Render (무료, 추천)
모든 학생이 인터넷으로 접속할 수 있어 수학여행 현장에 적합합니다.

1. 이 폴더(`server.js`, `package.json`, `public/`)를 GitHub 저장소에 올립니다.
2. https://render.com 가입 → **New → Web Service** → 그 저장소 연결
3. 설정:
   - Build Command: `npm install`
   - Start Command: `npm start`
4. 배포되면 `https://...onrender.com` 주소가 나옵니다. 이 주소를 학생들에게 공유하면 끝.

> 무료 플랜은 15분간 아무도 안 들어오면 잠들어서, 첫 접속이 30~50초 걸릴 수 있어요.
> 게임 시작 5분 전에 사회자가 먼저 접속해서 깨워두면 됩니다.

### B. 같은 와이파이에서 노트북으로 (인터넷 없이)
숙소 와이파이에 노트북과 학생 폰이 모두 연결돼 있고, 기기 간 통신이 막혀있지 않은 경우.

1. 노트북에 Node.js 설치 (https://nodejs.org, LTS 버전)
2. 이 폴더에서 터미널 열고:
   ```
   npm install
   npm start
   ```
3. 노트북의 내부 IP 확인 (예: `192.168.0.12`)
   - Windows: `ipconfig` / Mac: `ifconfig | grep inet`
4. 학생들은 폰 브라우저에 `http://192.168.0.12:3000` 입력해서 접속.

> ⚠️ 일부 숙소/학교 와이파이는 기기 간 통신(AP isolation)을 막아둬서 B 방식이 안 될 수 있어요.
> 미리 한 번 테스트해 보고, 안 되면 A(Render) 방식을 쓰세요.

### C. 로컬 테스트 (혼자 미리 연습)
```
npm install
npm start
```
브라우저 탭을 여러 개 열어서 (사회자 1 + 플레이어 여러 명) 흐름을 미리 연습해 보세요.

## 진행 팁
- 시작 전에 "밤에는 무조건 다 같이 폰 보고 채팅하기"를 규칙으로 정하세요. 그래야 연막이 효과 있어요.
- 마을 여론(참고용)은 단서일 뿐, **처형은 낮에 얼굴 보고 토론해서** 정하라고 안내하세요. 안 그러면 토론이 죽습니다.
- 밤 동안 사회자가 음악을 틀면 폰 진동/소리가 묻혀서 더 좋습니다.
- 첫 판은 특수직업 없이(마피아·경찰·의사·시민) 돌려보고, 익숙해지면 다음 판에 추가하세요.

## 기술 메모
- Node.js + Express + Socket.io (실시간 양방향 통신)
- 외부 의존성 최소화. 별도 DB 없이 메모리에서 방 상태 관리(서버 재시작 시 초기화).
